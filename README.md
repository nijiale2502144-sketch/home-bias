# Premier League Home Bias Analysis

This project analyzes the "home bias" phenomenon in the English Premier League (EPL) using data from the [TidyTuesday (2023-04-04)](https://github.com/rfordatascience/tidytuesday/tree/main/data/2023/2023-04-04) dataset.

For the full data analysis, methodology, and conclusions, please refer to the `report.pdf` file included in this repository.
